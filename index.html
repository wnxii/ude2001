The Gap

No existing solution bridges the gap between powerful citizen science platforms and the specific needs of Glasgow's biodiversity experts for efficient, AI-assisted local data management.

Proposed Solution

AI-Powered Biodiversity Data Integration Platform

A comprehensive four-part system that revolutionizes Glasgow's biodiversity monitoring

1 Automated data collection via iNaturalist API
2 AI-assisted pre-screening for accuracy
3 Expert verification with active learning
4 Customized dashboards for all stakeholders
ü§ñ Automated Intelligence

ML models analyze images and predict species accuracy (e.g., "User: Foxglove | Model: Foxglove (92%)")

üë©‚Äçüî¨ Expert Verification

Tinder-like interface for quick expert review with active learning that improves over time

üìä Decision Support

Real-time dashboards with AI alerts for decision-makers (e.g., invasive species tracking)

üåç Public Engagement

Citizen-friendly portal with photo ID, location-based exploration, and gamification

Result: Transform quarterly manual processing into real-time, scalable biodiversity monitoring
