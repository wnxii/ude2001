Public-Facing Interface
5

Lack of Built-in Camera

"The issue with the design of this page was that users will have to take a photo of the animal with their camera and upload the file onto the website to identify. This two step process is cumbersome and has alot of friction, interrupting the flow of the process. Hence, allowing the option to access the camera and take pictures directly will reduce user friction."

- User Testing Participant

Current Design
Current Two-Step Process Issues
Step 1: External Photo Capture

Users must use their device's camera app to take photos separately from the identification process

Step 2: File Upload Process

Navigate back to the website, find the captured image, and upload through file browser

‚ö†Ô∏è User Friction Points
  • Context switching between camera app and website
  • Potential loss of identification momentum
  • Additional navigation steps in file system
  • Higher probability of user abandonment
üí° Solution Implemented
‚úì

Integrated Camera Access: Direct camera integration within the web application

  • Added direct camera access button alongside file upload option
  • Eliminated context switching between apps
  • Streamlined one-step photo capture and identification process
  • Reduced user friction and improved workflow continuity
  • Maintained file upload option for users with existing photos

Tutor Feedback Session

In-depth insights and feedbacks from our research tutor

üí° Feedback & Insights Mentioned
Disagree Functionality Concerns

Questioned the purpose of the "disagree" button when neither of the AI's suggestions are correct. This could create unnecessary cognitive load on experts who frequently need to disagree with incorrect identifications.

Public Input Quality Issues

Concerns were raised about the quality of images uploaded by the public, including blurry or unclear pictures, and people uploading images for non-scientific purposes.

Low-Confidence Image Filtering

Feedback highlighted that automatically filtering out images with low AI confidence might unintentionally discard images of new or rarely seen species, potentially missing important scientific discoveries.
